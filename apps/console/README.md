# Console App

This is the Next.js operations console.

## Structure

- `app/` App Router routes
- `components/ui/` shadcn-compatible UI component path
- `lib/` shared UI utilities

## Run

From repository root:

```bash
npm install
npm run dev:console
```

Open:
- `http://localhost:3000/`
- `http://localhost:3000/demo`
